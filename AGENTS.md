# AI Skills Router

This repo is the single source of truth for all AI agent skills.
Use symlinks to connect each IDE to the skills here.

## Workflow

- `workflow/code-review` — PR review, diff review, merge readiness
- `workflow/question-details` — Requirement clarification before implementation
- `workflow/requirements-plan-design-thinking` — Feature design, architecture trade-offs, co-creation

## Backend

- `backend/write-postgres-sql` — PostgreSQL query generation, optimization, schema-aware SQL
